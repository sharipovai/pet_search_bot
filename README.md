**PetSearch Bot** — это Telegram-бот со встроенным искусственным интеллектом, созданный для помощи в поиске потерянных и найденных животных (кошек и собак). 

Вместо ручного просмотра сотен объявлений, бот использует передовые модели компьютерного зрения (YOLOv8 + DINOv2) для распознавания питомцев и векторную базу данных для мгновенного нахождения визуальных совпадений.

## ✨ Ключевые возможности

- 📝 **Разделение на статусы:** Пользователь может оставить заявку о потере животного или сообщить о найденном.
- 🏙 **Фильтрация по городу:** Поиск происходит в рамках указанного населенного пункта.
- ⚡ **Real-time Мэтчинг:** Как только новая заявка попадает в базу, бот мгновенно ищет совпадения среди противоположных статусов (Потерян <-> Найден) и присылает уведомление с фото и контактами.
- 🤖 **Защита от "мусора":** Бот автоматически определяет наличие животного на фото. Если прислать фото машины или пейзажа, бот попросит загрузить корректное изображение.
- 📞 **Связь напрямую:** Удобный обмен контактами для связи между хозяином и нашедшим.

## 🧠 Как это работает (ML Pipeline)

Проект использует современный подход к поиску по изображениям (Content-Based Image Retrieval):

1. **Детекция и Сегментация (`YOLOv8n-seg`):**
   Когда пользователь загружает фото, YOLOv8 находит на нём кошку или собаку, вырезает питомца по пиксельной маске (сегментация) и заливает фон черным цветом. Это критически важно, чтобы нейросеть не обращала внимание на траву, асфальт или ковры при поиске.
2. **Получение эмбеддингов (`DINOv2` от Meta):**
   Очищенное от фона изображение передается в DINOv2. Модель формирует математический вектор (эмбеддинг) размерностью 384 признака, который описывает текстуру шерсти, пятна, форму ушей и другие уникальные черты животного.
3. **Векторный поиск (`PostgreSQL` + `pgvector`):**
   Эмбеддинг сохраняется в базу данных. Для поиска похожих животных используется вычисление **косинусного расстояния (Cosine Distance)** на уровне SQL-запроса. Это позволяет находить похожих питомцев за миллисекунды.

## 🛠 Стек технологий

* **Язык:** Python 3.11+
* **Бот-фреймворк:** PyTelegramBotAPI (асинхронный режим) / aiogram
* **База данных:** PostgreSQL + расширение `pgvector`
* **ORM:** SQLAlchemy 2.0 + asyncpg
* **Machine Learning:**
  * `PyTorch` & `Transformers` (HuggingFace)
  * `ultralytics` (YOLO)
  * `OpenCV` & `Pillow` (обработка изображений)
* **Инфраструктура:** Docker, Docker Compose
